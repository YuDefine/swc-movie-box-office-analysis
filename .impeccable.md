# Design Context — 陽光女子合唱團票房儀表板

## Concept: Golden Hour Cinema
The dashboard tells the story of a film's triumph through data.
Not a cold analytics tool — a warm, narrative experience.

## Visual Identity
- **Palette:** Amber/gold core, emerald for growth, violet for analysis, rose for contrast
- **Background:** Subtle warm tints, alternating section shading for rhythm
- **Cards:** Clean with subtle shadow, hover elevation
- **Dark Mode:** First-class — deep warm blacks, glowing accents

## Typography
- **Font:** Huninn 粉圓體 — friendly, local, approachable
- **Hero numbers:** Display scale (text-5xl+), tight tracking
- **Section headers:** text-xl, bold, with colored dot accent
- **Data:** Tabular numerals for alignment

## Layout Principles
1. Dramatic hero — the triumph is the first thing you see
2. Stats as inline horizontal bar (not separate cards)
3. Achievement milestones as compact side-by-side cards
4. Charts with clear hierarchy — cumulative is featured
5. Sections alternate subtle background tints for rhythm
6. Generous whitespace between major sections

## Motion
- Scroll-reveal (IntersectionObserver, one-shot)
- Staggered stat entrances (60ms intervals)
- Number counting animation (800ms, ease-out cubic)
- ProgressCircle fills from 0
- Hero entrance with subtle scale + fade
- All respect prefers-reduced-motion

## Tech Stack
- Nuxt 4 + @nuxt/ui 4.4 + nuxt-charts 2.1
- Tailwind CSS (via @nuxt/ui)
- No additional dependencies for animation
